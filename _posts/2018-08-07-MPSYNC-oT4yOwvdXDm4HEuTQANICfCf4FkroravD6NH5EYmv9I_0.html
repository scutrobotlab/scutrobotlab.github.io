目前，实验室分组为机械组、电控组、视觉组、宣传组、运营组。

下面按组别给出入门学习建议。加粗部分为必须完成的事项，其他内容大致了解即可，进实验室之后会有相关培训。


机械组


主要负责机器人机械结构的设计加工。包括设计和三维建模，计算机辅助有限元分析，机械零件的加工装配，机械体冲击测试、维护和改进。有兴趣的同学可以报名加工中心的Solidcam培训。

你将学到：机械原理、机械工程材料、机械制图、力学、机械加工装配等知识。

入门必备技能：SolidWorks、机械设计和加工基础知识、3D打印机原理

请先安装 SolidWorks，并掌握基本操作。


电控组


主要负责机器人电子线路和程序控制。包括电路设计和制作，嵌入式主控单元的软硬件开发，通信接口的封装和对接，各类电驱设备的选型和驱动，机器人算法控制。目前实验室主要选用芯片为STM32系列，主要程序语言为C。

你将学到：电子电路、嵌入式开发，PCB设计、传感器、计算机通信网络、自动控制算法等知识。

入门必备技能：Keil μVision IDE 和 Git （程序控制方向）、Altium Designer 和 焊板子 （硬件开发方向）

请先了解单片机基础知识，了解 Git 的使用。

Git 是一个开源的分布式版本控制软件，除了常见的命令行外，你也可以在官网的GUIs列表中挑选合适的图形界面。官方页面：https://git-scm.com/


视觉组


主要负责机器人所搭载计算机系统的开发，使机器人具备感知功能。具体任务包括图像处理、神经网络、SLAM、基于ROS系统的开发等。主要程序语言为C++。

视觉组需要有良好的数学功底，掌握图像处理算法，Linux系统开发等知识。

入门必备技能：OpenCV、Linux、Git

请先安装Visual Studio，配置好OpenCV的环境，并了解Git的使用。学有余力的同学可安装Ubuntu系统，研究一下各种神经网络。

Visual Studio 2017 的 Community 版本已经免费。官方下载地址：https://www.visualstudio.com/zh-hans/downloads/

OpenCV是一个开源的视觉识别系统。在官方页面的右侧点击Tutorials有完整入门教程。官方页面：http://www.opencv.org/

Ubuntu是一个基于Linux的开源操作系统。官方页面：http://cn.ubuntu.com/

Git是一个开源的分布式版本控制软件，除了常见的命令行外，你也可以在官网的GUIs列表中挑选合适的图形界面。官方页面：https://git-scm.com/


宣传组


主要负责宣传推广和策划各类校内外活动。制作实验室宣传品，包括文案、视频、周边，参加各类评优推选（十佳团队、感动华园等），策划会展和活动（琶洲国际会展、广州科普开放日、飞手训练营活动等）和各类表演（电视台节目和校内活动等）。当技术组水平不够的时候化身为战略忽悠组，依靠强大的吹水能力震慑对手。

你将学到：Photoshop、After Effect、Prime，摄影和录像技巧，优秀的文案、公众号排版（呵呵）。

你将拥有：GoPro运动相机、OSMO手持云台相机、各种型号无人机、单反

请至少观看一场Robomaster全国赛视频，并以你最擅长的方式，对华工的其中一场比赛做一个宣传，比如文章、视频、图片，任何你喜欢的形式。

RoboMaster全国赛视频集锦：http://v.qq.com/detail/m/mr3x0swpumstoyo.html


运营组


主要负责商业运作、资金管理和财务报表的制作。具体工作包括申请商业赞助，将技术组产品进行商业包装，与企事业单位对接，管控和审批队员开支，核对报账发票，制作月季度报表。运营组参加全国大学生机器人创业大赛、挑战杯、互联网+等商业比赛。

你将学到：会计学基础知识、发票报账制度

入门必备技能：撰写商业计划书、Excel
